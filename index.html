RU_AI_unlocker

Open-source решение для обхода региональных ограничений популярных нейросетей и AI-сервисов. Получите доступ к ChatGPT, Midjourney, Claude и другим платформам из России.

Оптимизированные подключения без потери скорости

ChatGPT, Midjourney, Stable Diffusion, Claude и другие

Быстрая установка

Клонирование репозитория

git clone https://github.com/F1NDLE/RU_AI_UNLOCKER.git

Установка зависимостей

cd RU_AI_UNLOCKER
pip install -r requirements.txt

Запуск скрипта

python ru_ai_unlocker.py

Настройка браузера

После запуска следуйте инструкциям в терминале для настройки прокси в браузере
